# WeChat Mini Program Project Context

## Project Overview
This is a WeChat Mini Program project built with TypeScript for managing coffee bean data.

## Technology Stack
- **Language**: TypeScript (strict mode)
- **Framework**: WeChat Mini Program
- **Module System**: CommonJS
- **Target Runtime**: ES2020

## Key Features
- Data table with filtering capabilities
- Country and type filters sourced from API
- Dynamic API hostname configuration based on environment
- Responsive table with horizontal scrolling
- Detailed view modal for data records

## Project Structure
- `/miniprogram`: Main source code
- `/miniprogram/pages`: Individual page components
- `/miniprogram/services`: API service layer
- `/miniprogram/types`: TypeScript type definitions
- `/miniprogram/config`: Configuration files

## Environment Configuration
- **Development (Computer)**: localhost
- **Development (Device)**: Configurable IP address
- **Production**: Custom domain

## Key Components
- **Data Page**: Main data table with filtering
- **Coffee Service**: API communication layer
- **Configuration**: Dynamic environment settings
